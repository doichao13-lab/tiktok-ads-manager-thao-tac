⚙ Cài đặt kết nối TikTok
📖 Hướng dẫn: Tạo app tại TikTok for Business → Apps, sau đó lấy Access Token với các quyền: campaign.read, campaign.write, adgroup.read, adgroup.write, report.read, advertiser.read.
Lưu ý: Nếu gặp lỗi CORS, điền proxy bên dưới.
Xác thực API
Lấy từ: TikTok for Business → Tài khoản → Apps → Your App → Access Token
Nếu gặp lỗi CORS khi chạy trên GitHub Pages, điền: https://corsproxy.io/?url=
Tài khoản quảng cáo
Nhập ID các tài khoản quảng cáo muốn quản lý. Để trống để tự động lấy tất cả.
