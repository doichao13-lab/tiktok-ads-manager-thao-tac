⚡ Demo Mode
💡 Cách này dành cho bạn — bạn đã có App ID và Secret trên TikTok Business Portal. Tool sẽ tự tạo link đăng nhập → bạn ủy quyền → tool tự lấy token.
1
Nhập App ID và Secret
Copy từ trang App trong TikTok Business Portal
2
Chọn Redirect URI đã đăng ký trong App
TikTok chỉ chấp nhận redirect URI đã được đăng ký trong App của bạn.
Nhìn vào trang App → mục "Advertiser redirect URLs" → chọn 1 trong các URL đó:
Dán chính xác 1 trong các URL ở mục "Advertiser redirect URLs" trong App của bạn.
3
Copy auth_code từ URL sau khi redirect
Sau khi nhấn Authorize, TikTok redirect sang trang đã đăng ký (VD: seongon.com). URL trên thanh địa chỉ sẽ có dạng:
https://seongon.com/?auth_code=AbCdEf123XyZ&state=hung_tiktok_tool
👆 Copy đúng phần sau auth_code= (không lấy &state=...)
4
Token đã sẵn sàng — Lưu và kết nối
Token sẽ tự động điền sau khi đổi thành công. Nhấn "Lưu & Kết nối" bên dưới.
Dùng localhost:9090 — chạy PowerShell proxy bên dưới, đáng tin cậy nhất.
▶ Script PowerShell chạy proxy cục bộ (click để xem)
Add-Type -AssemblyName System.Web $listener = New-Object System.Net.HttpListener $listener.Prefixes.Add('http://localhost:9090/') $listener.Start() Write-Host "Proxy TikTok dang chay: http://localhost:9090/" -ForegroundColor Green Write-Host "Giu PowerShell nay mo trong khi dung tool." while ($listener.IsListening) { $ctx = $listener.GetContext() $req = $ctx.Request; $res = $ctx.Response $res.Headers.Add('Access-Control-Allow-Origin','*') $res.Headers.Add('Access-Control-Allow-Headers','*') $res.Headers.Add('Access-Control-Allow-Methods','GET,POST,OPTIONS') if ($req.HttpMethod -eq 'OPTIONS') { $res.StatusCode=200; $res.Close(); continue } try { $targetUrl = [System.Web.HttpUtility]::UrlDecode($req.QueryString['url']) $token = $req.Headers['Access-Token'] $hdrs = @{'Access-Token'=$token} if ($req.HttpMethod -eq 'POST') { $body = (New-Object System.IO.StreamReader($req.InputStream,[System.Text.Encoding]::UTF8)).ReadToEnd() $wr = Invoke-WebRequest -Uri $targetUrl -Method POST -Headers $hdrs -Body $body -ContentType 'application/json' -UseBasicParsing } else { $wr = Invoke-WebRequest -Uri $targetUrl -Method GET -Headers $hdrs -UseBasicParsing } $json = $wr.Content } catch { $json = "{`"code`":-1,`"message`":`"$($_.Exception.Message)`",`"data`":null}" } $bytes = [System.Text.Encoding]::UTF8.GetBytes($json) $res.ContentType = 'application/json; charset=utf-8' $res.ContentLength64 = $bytes.Length $res.OutputStream.Write($bytes,0,$bytes.Length) $res.Close() }
⚠ Dùng tab này nếu bạn đã có sẵn Access Token (không phải Secret). Nếu chưa có token, hãy dùng tab "App ID + Secret".
Lấy từ: TikTok Business Portal → My Apps → App của bạn → tab Auth → Access Token
